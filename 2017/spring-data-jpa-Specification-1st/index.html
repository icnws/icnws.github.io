这个工具类定义了一些基本的查询，包括模糊匹配（containsLike）,数值区间（isBetween）以及枚举类参数匹配（enumMatcher），这些也是在查询中常用的方法，在查询的时候，通过Specifications进行调用组织参数，即可获得接口Specification的实例，然后用于查询。
